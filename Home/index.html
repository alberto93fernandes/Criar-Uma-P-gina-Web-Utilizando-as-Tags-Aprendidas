Estrutura básica do HTML

Voltar ao topo

Entender a estrutura de um website é bem simples, por estar dividida basicamente em duas partes, head e body. Onde cada um possue um comandos que lhe são próprios ou indicados para padronizá-lo. Essa construção básica que parte, inclusive da nomeação do arquivo html como "index" convenciona um padrão a fim de facilitar a identificação no presente e futuro.

Falando Sobre Tags

Voltar ao topo

Tags são marcadores instruem o navegador sobre cada informação ou elemento que é adicionado a ele, que (ainda) sem o auxílio do CSS, permite ao navegador estabelecer um certo estilo organizacional. Com isso, os navegadores, interpretação as marcações feitas pelo desenvolvedor e devolvem ao usuário o comando que foi atribuido.

Atributos das Tags

Voltar ao topo

Além das tags, temos o que chamamos de atributos, que nada mais são algumas configurações que são inclusas dentro de tags para que elas apareçam de uma forma diferente no navegador. São propriedades que uma tag pode ter, podendo ser globais (genéricas) ou específicas de cada tag.

Textos

Voltar ao topo

Os textos também possuem tags específicas para serem exibidas no navegador, como as tags H1 ao H6, que compreendem os tamanhos dos títulos e subtítulos, sendo que o H1 é o maior tamanho e o menor o H6. Os parágrafos, os textos que carregam o conteúdo propriamente dito, possuem uma tag específica para serem definidos como parágrafo que é < p > e com elas podemos escrever o nosso texto livremente e inserindo tags no meio do parágrafo, se assim for o nosso objetivo também.

Listas ordenadas e Não ordenadas

Voltar ao topo

Uma das funções mais usadas e presentes em um website são as listas, divididas em "ordenadas" e "não ordenada". Como os nomes já sugerem, onde cada uma também possuem tags específicas para serem usadas e torna-las funcionais, pois necessitam das "tags filhas".
